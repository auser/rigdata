What

This is a demo of how to use Storm with jRuby

.

Requirements

  • Redis
  • Java
  • rvm/rbenv

Install


        rvm install jruby
        gem install bundler
        bundle install

Run the redis


        redis-server

Run the web server


        bundle exec puma

Run Storm


        ./bin/run_topology.sh lib/backend/topologies/twitter.rb
